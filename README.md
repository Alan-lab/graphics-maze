# graphics-maze
图形学大程，迷宫游戏

#初步设想
1. 记录时间和收获的金币。--何
2. 金币随机出现，大约每10个空地一块金币--何
3. 添加地图卷轴，在距离起始格一定距离的地方随机出现。找到地图卷轴之后，右上方出现俯视图，标记主角当前位置和记录已经探索过的地方。--何（卷轴设计）
4. 在墙面添加壁灯。--陈
5. 添加地面和天花板的纹理。--徐
6. 添加入口和出口的装饰。--杨，苑
